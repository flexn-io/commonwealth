# @flexn/plugins

Example usage:

`./renative.json`

```
{
    "paths": {
        "pluginTemplates": {
            "flexn": {
                "npm": "@flexn/plugins",
                "path": "./pluginTemplates"
            }
        }
    },
    "plugins": {
        "plugin-name": "source:flexn"
    }
}
```
